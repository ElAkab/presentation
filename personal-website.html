Projects

Interactive Calendar

This interface is perfect for scheduling one or more dates that work for everyone.

It allows you to freely choose as many dates as needed and stores them automatically. One of the highlights of this project is that holidays and vacation days are visually distinguished with a unique style, making them easier to identify at a glance.

HTTP Code Training

This project was designed to help memorize common HTTP/HTTPS status codes.

On the left side, buttons are available for each status code. Once a button is clicked, it gets disabled to avoid repetition. On the right side, the corresponding explanation appears, with an option to hide or reveal it. This makes the tool interactive, simple to use, and very effective for practice.

Password Generator
(Sign-Up Form)

A sign-up form with a built-in password generator.

This project provides users with a modern and secure way to create strong passwords instantly. The form includes a password generator that suggests complex combinations of letters, numbers, and special characters, significantly enhancing account security. Users can also confirm the generated password to ensure it matches before submitting the form, offering a smooth and reliable sign-up experience. This feature demonstrates both usability and best practices in user authentication design.
